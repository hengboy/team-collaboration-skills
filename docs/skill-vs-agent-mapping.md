# Skill vs Agent 映射规则

## 定义

- `SKILL.md`：完整角色说明与事实源
- `AGENT.md`：平台适配后的精简派生物

## 映射原则

| 维度 | SKILL.md | AGENT.md |
|------|----------|----------|
| 角色定位 | 完整 | 必须一致 |
| 适用场景 | 完整 | 可精简 |
| 必须输入 | 完整 | 必须一致 |
| 可选输入 | 完整 | 必须一致 |
| 输出文件 | 完整 | 必须一致 |
| 执行规则 | 完整 | 必须一致 |
| 质量检查 | 完整 | 必须一致 |
| 下一步建议 | 完整 | 必须一致 |
| 示例与长模板 | 可包含 | 可省略 |

## 当前同步方式

当前通过每个文件中的 `## 核心契约（供 AGENT 派生）` 做精确对比。

## 何时需要同步

- 角色职责变化
- 输入输出变化
- 路径规则变化
- 质量门槛变化
- 下一步流转变化

只改示例或长解释时，可以不改 agent。
