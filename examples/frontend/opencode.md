# frontend - OpenCode 使用示例

## 使用方式

```bash
# 1. 启动 OpenCode
opencode

# 2. 加载 Skill
skill(name: frontend)

# 3. 描述任务（用@引用文件）
请完成具体任务。

## 相关文档
@.collaboration/features/{feature-name}/prd.md
@.collaboration/features/{feature-name}/api.yaml
```

**无需脚本** - OpenCode 会自动读取 `@` 引用的文件。
